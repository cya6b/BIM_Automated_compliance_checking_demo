### 项目介绍
BIM模型审查系统的web网页端

支持gfc格式模型文件的智能审查及审查结果查看校核

### 编译与运行
#### 编译环境
 windows 7 及以上版本操作系统
 node v16.17.0

##### 安装基础依赖组件
node
npm

#### 编译步骤
``` bash
# install dependencies
npm install

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

#### 运行环境
Windows 7 及以上版本操作系统
chrome 83及以上版本浏览器

#### 运行步骤
``` bash

# serve with hot reload at localhost:8080
npm run dev
```
